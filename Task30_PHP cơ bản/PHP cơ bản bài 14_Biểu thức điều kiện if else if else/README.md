### Biểu thức điều kiện if else if else:

- Ví dụ 1: 

```javascript
<?php
	$dayOfWeek = 2; 
	if ($dayOfWeek == 2){
		echo 'Monday';
	} else if ($dayOfWeek == 3){
		echo 'Tuesday';
	} else if ($dayOfWeek == 4){
		echo 'Wednesday';
	} else if ($dayOfWeek == 5){
		echo 'Thursday';
	} else {
		echo ' sorry, please try again !!!';
	}
?>
```
