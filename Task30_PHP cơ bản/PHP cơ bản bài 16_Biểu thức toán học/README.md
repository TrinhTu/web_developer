### Biểu thức toán học:

- Các biểu thức toán học cơ bản: cộng (+), trừ (-), nhân (*), chia (/), chia lấy dư (%).

- Ví dụ:

```javascript
<?php
	$i = 5;
	$j = 10;

	$x = $i + $j;
	$y = $j - $i;
	$k = $i * $j;
	$z = $j / $i;
	$u = $j % $i;

	echo $x,', ',$y,', ',$k,', ',$z,', ',$u;
?>
```
