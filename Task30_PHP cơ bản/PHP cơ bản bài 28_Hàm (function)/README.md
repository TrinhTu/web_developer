### Hàm (function):

- Ví dụ:

```javascript
<?php
	function hello_world(){
		echo 'Hello World !';
		echo ' . I\'m a student<br />';
	}

	function sample(){
		for ($i = 0; $i < 100; $i++){
			echo 'This is the ' .$i.'<br />';
		}
	}
	hello_world();
	sample();
?>
```
