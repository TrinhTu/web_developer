### Nhúng PHP vào trang HTML:

- PHP là ngôn ngữ nhúng trực tiếp vào HTML bằng cách lưu lại file với đuôi `.php`

- `echo` là hàm in các chuỗi dạng text ra ngoài màn hình

```javascript
<?php 
	echo 'Hoc PHP co ban';
	echo '<h1>Tieu de 1</h1>';
	echo '<h2>Tieu de 2</h2>';
?>
```
