### Sử dụng echo,print:

- echo dùng để đưa 1 chuỗi, biến, hằng... ra màn hình

```javascript
<?php
	$a = 'XIN CHAO CAC BAN';
	echo $a;

?>
```

- print cũng tương tự như echo là in giá trị ra màn hình
